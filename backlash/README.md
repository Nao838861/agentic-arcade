# BACKLASH

縦スクロールシューティング。

## 操作
- 移動: ← → ↑ ↓ または WASD
- ボム: SPACE（ゲージMAX時）
- ミュート: M

## 仕様
- ゲージはアイテム取得で上昇。閾値で弾が増える（1way → 2way → 3way）
- MAX (208): SPACEでボム発動。プレイヤー周辺の一定範囲を一掃
- 中ボス（ティール六角形）: 撃ち漏らすと逃走。倒すと多めにアイテムドロップ
- ボス（ピンク八角形）: HPバー付き、長い滞在
- 被弾はゲージで身代わり、ゲージ0で死亡
