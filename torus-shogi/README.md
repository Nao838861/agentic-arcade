# トーラス将棋 (Torus Shogi)

上下・左右がつながったトーラス（ドーナツ）盤の将棋。`index.html` をブラウザで開くだけで、COM と対戦できる（依存なし・単一ファイル）。

▶ **遊ぶ: https://nao838861.github.io/agentic-arcade/torus-shogi/**

## トーラス将棋とは
盤に「端」がない。上の端を越えた駒は下の端から、左の端を越えた駒は右の端から現れる。駒を選ぶと利きが反対側に回り込むのが見える。

## 通常将棋との差
- **初期配置**: 通常将棋を各陣営とも敵方向へ1段寄せた配置（上下ラップによる玉の密着を回避）。
- **成りゾーン**: 移動後の相手陣3段。任意で成れる。
- **飛・角・香**: ラップして進み、最初にぶつかった駒で止まる。一周すると自分の升に戻るので自分自身は取れない（＝空いた筋・段・斜めなら全升に利く）。このため遠い飛角香が桁違いに強く、合駒が効きにくい。
- **二歩**は禁止。行き所のない駒・打ち歩詰め・千日手の制約は無し（簡易版）。

## 遊び方
- 自分の駒をクリック → 行ける升が緑（移動）/赤リング（駒を取る）でハイライト。
- ハイライトをクリックで着手。成れる場合はダイアログで選択。
- 持ち駒をクリック → 打てる空き升がハイライト。
- 「強さ」「手番」を選んで「新規対局」。「待った」で自分の手番まで戻す。
- **「ループ表示」**で本来の9×9（赤枠）の外側にループ先を表示。盤の端がどうつながっているか一目で分かる。

## COM
αβ法（negamax）＋ 反復深化 ＋ 静止探索 ＋ null-move枝刈り ＋ 手順並べ替え（MVV-LVA・キラー手）。評価は駒得＋玉周りの被利き（端ボーナスは使わない）。
